# Requirements

## Functional Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| REQ-001 | Framework must persist state in `.context/` directory | Must Have |
| REQ-002 | Framework must provide spec templates for features | Must Have |
| REQ-003 | AI must read state files at session start | Must Have |
| REQ-004 | AI must require explicit approval before implementation | Must Have |
| REQ-005 | Framework must support task tracking with dependencies | Should Have |
| REQ-006 | Framework must support parallel workstreams | Should Have |
| REQ-007 | Framework must auto-generate kanban board view | Should Have |

## Non-Functional Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| NFR-001 | All files must be plain Markdown | Must Have |
| NFR-002 | Framework must work with any AI assistant | Must Have |
| NFR-003 | Starter kit must be copy-pasteable | Must Have |
| NFR-004 | Documentation must be self-explanatory | Should Have |
